## 시놉시스
<i>주인공은 아주 유연한 고양이 3마리를 키우고 있었다.<br>
잘 기르다 보니 33마리로 불어난 고양이들.<br>
어느날 집에 돌아와보니 고양이가 단 한 마리도 없었고…<br>
#### 고양이가 다 어디로 갔을까!?</i>

<br><br>

## 게임 목표
1. 고양이를 전부 다 찾아서 모든 스테이지 클리어<br>
2. 시간 안에 더 많은 고양이를 찾는 사람이 우승 (만약 동점자가 있다면 걸린 시간으로 판별)
